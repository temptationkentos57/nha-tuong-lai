# Nha Tương Lai

## Giới thiệu
Nha Tương Lai là một nền tảng giúp các freelancer và doanh nghiệp tại Việt Nam kết nối và hợp tác dễ dàng trong môi trường làm việc từ xa. Ứng dụng cung cấp các công cụ cần thiết để quản lý dự án, giao tiếp, và thanh toán, giúp tối ưu hóa quy trình làm việc.

## Tính năng chính
- **Kết nối freelancer và doanh nghiệp**: Dễ dàng tìm kiếm và tuyển dụng nhân viên tự do.
- **Quản lý dự án**: Thao tác với các dự án, giao nhiệm vụ và theo dõi tiến độ.
- **Giao tiếp**: Tích hợp chat và video call để dễ dàng trao đổi thông tin.
- **Thanh toán trực tuyến**: An toàn, nhanh chóng và dễ dàng.

## Cài đặt
1. Clone repository này.
2. Cài đặt các phụ thuộc bằng `npm install`.
3. Chạy ứng dụng bằng `npm start`.

## Công nghệ sử dụng
- React cho frontend
- Node.js cho backend
- MongoDB cho cơ sở dữ liệu
